# Privacy Policy

This browser extension does not collect, store, or transmit any user data.

## Data Collection

- **No personal information** is collected
- **No browsing history** is tracked
- **No analytics or telemetry** is sent
- **No data** is shared with third parties

## How It Works

This extension operates entirely locally within your browser. It only reads search query information from ChatGPT pages to display in the browser toolbar. All processing happens on your device and nothing leaves your browser.

## Contact

If you have any questions about this privacy policy, please open an issue on the GitHub repository.
